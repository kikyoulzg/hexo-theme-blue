# Hexo Theme Green

A super simple theme for Hexo, fork on even.

[中文文档](./README_CN.md)

## Preview

You can visit my [blog](https://blog.rekols.com/)

## Installation

```
$ npm install hexo-renderer-scss --save
$ git clone https://github.com/rekols/hexo-theme-green.git themes/green
```

## Update

```
$ cd themes/green
$ git pull
```
