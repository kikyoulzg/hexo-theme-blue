# Hexo 帅气蓝主题

超级简约的主题, 基于Green（ even）.

## 预览

具体样式可以访问我的博客： [https://kikyoulzg.github.io](https://kikyoulzg.github.io)

## 安装

```
$ npm install hexo-renderer-scss --save
$ git clone https://github.com/kikyoulzg/hexo-theme-blue.git themes/blue
```

## 更新

```
$ cd themes/blue
$ git pull
```
